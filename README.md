# Project Portfolio Overview

## 1. NYC Public School Data Dashboard  
Interactive Dash application for exploring NYC student demographic data across boroughs and years.  
**Key Skills:** Python, Dash, Plotly, Pandas, API integration, callback architecture  

## 2. NYC Public School Machine Learning Project  
Machine learning modeling in R to forecast and classify NYC school performance ratings using multiple model families.  
**Key Skills:** R, GLM, Poisson models, decision trees, feature engineering, model diagnostics  

## 3. LLA to ECEF Geospatial Conversion Challenge  
Engineering-heavy coding challenge implementing precise geodetic coordinate transformations, velocity estimation, interpolation, and physical verification.  
**Key Skills:** Python, geospatial math, pyproj, numerical methods, error tolerance checks  

## 4. Digital Platform Engagement Time Series Forecasting  
Multi-year forecasting project analyzing student and teacher digital engagement patterns across NYC schools using PySpark and SARIMAX.  
**Key Skills:** PySpark, time series modeling, anomaly detection, Tableau/Power BI reporting
